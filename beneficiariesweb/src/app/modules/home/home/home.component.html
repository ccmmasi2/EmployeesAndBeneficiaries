Employee's beneficiaries

        
In the upper side you will find a Select control:

        
1 - Select Employees option to interact with the Employee's options
 
        
         - Create, update, delete and list employees

        
         - Select one Employee and add Beneficiaries to the list

        
         - Create, update, delete and list Beneficiaries for each Employee

        
2 - Select Beneficiaries option to interact with the Beneficiaries' options

        
         - Create, update, delete and list Beneficiaries for each Employee
